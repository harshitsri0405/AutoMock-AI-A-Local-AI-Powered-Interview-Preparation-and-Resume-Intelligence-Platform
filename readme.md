# AutoMock AI

### A Local AI-Powered Interview Preparation and Resume Intelligence Platform

## Overview

**AutoMock AI** is an AI-powered platform designed to help students and job seekers prepare for interviews more effectively. It combines **topic-based interview question generation**, **resume-based personalized question generation**, **resume analysis with improvement suggestions**, **AI-assisted resume building**, and **answer evaluation** into a single system.

The platform analyzes a user’s **skills, resume content, projects, and GitHub repository context** to generate relevant interview questions and provide resume intelligence for better placement preparation.

## Key Features

* **Topic-Based Question Generation** for skills like Python, Java, C++, SQL, Aptitude, DBMS, and more
* **Resume-Based Personalized Interview Questions** based on skills, projects, and technical background
* **Resume Analysis & Improvement Suggestions** to identify missing sections and improve ATS readiness
* **AI Resume Builder** to help create structured and professional resumes
* **Answer Evaluation** by comparing user responses with ideal answers
* **GitHub Repository Link Integration** for project-aware interview question generation

## Tech Stack

* **Python**
* **Streamlit**
* **Machine Learning / NLP**
* **Local ML / LLM workflow**
* **CSV-based skill datasets**

## Purpose

AutoMock AI aims to provide a **smart, adaptive, and end-to-end interview preparation experience** by combining **resume intelligence, mock interview practice, and answer evaluation** in one platform.
